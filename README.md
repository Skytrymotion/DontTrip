# Dont Trip

*Makes you Randomly Trip and Fall*

!!Client Sided!!


## Features
- Trip and Fall
- Take Damage (configurable on/off & damage amount)
- Chance of Tripping (configurable)

[Version 1.2.0+]
- Settings Sync with Host

## Notes
- Client Sided, not everyone needs to install it

## Requirements
- BepInEx LTS (5.4.22)
